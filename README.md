# 📊 Data Jobs Dashboard 

## Introduction
This dashboard was created for **Job Seekers, Career Transitioners, and Workforce Analysts** to address a common challenge: information about the data job market is fragmented and difficult to interpret.  

Using a real-world dataset of **2024 data-related job postings** (titles, salaries, benefits, and locations), this project consolidates insights into a single, easy-to-use interface. Users can explore **market trends, compensation benchmarks, and role-specific details** to make informed career and hiring decisions.

## Skills Showcased
This project demonstrates a wide range of Power BI capabilities:

- ⚙️ **Data Transformation (ETL) with Power Query**  
  Cleaned and shaped raw job posting data, handled blanks, standardized data types, and created calculated columns.

- 🧮 **Measures & KPIs**  
  Built implicit and explicit measures to calculate Median Salaries, Job Counts, and Work Condition percentages.

- 📊 **Core Visualizations**  
  Column, Bar, Line, Area, and Scatter Charts to compare job counts, salaries, and track trends over time.

- 🗺️ **Geospatial Analysis**  
  Map visuals to show global job distribution and density.

- 🔢 **KPI Indicators & Tables**  
  Cards for quick metrics and tables for sortable, detailed job statistics.

- 🎨 **Dashboard Design**  
  Intuitive layout with clear navigation, combining overview and drill-through pages.

- 🖱️ **Interactive Reporting**  
  - **Slicers**: Filter by job title, region, or platform.  
  - **Buttons & Bookmarks**: Seamless navigation between views.  
  - **Drill-Through**: Move from high-level summaries to detailed role-specific insights.

## Dashboard Overview
The report is split into **two main pages**:

### Page 1: High-Level Market View
Your mission control for the data job market.  
- Total job count, median salaries, and top job titles.  
- Job trends over time.  
- Comparative salary analysis (hourly vs yearly).  
- Role distribution across Data Engineers, Analysts, Scientists, and more.  

### Page 2: Job Role Drill-Through
The deep-dive view.  
- Salary ranges (yearly/hourly) with averages.  
- Work-from-home %, degree requirement %, health insurance %.  
- Global job distribution map.  
- Hiring platforms breakdown (LinkedIn, Indeed, ZipRecruiter, Jooble, BeBee).  
- Job schedule type (full-time vs contractor).

## Conclusion
This dashboard demonstrates how **Power BI can transform raw job posting data into actionable insights**. It empowers users to slice, filter, and drill through data to understand the job market, benchmark compensation, and plan career moves or recruitment strategies effectively.
